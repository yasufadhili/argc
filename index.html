Examples

Fibonacci Sequence

func fibonacci(n) {
if n <= 1 return n
return fibonacci(n-1) + fibonacci(n-2)
}

A classic recursive implementation of the Fibonacci sequence.

String Manipulation

func reverse_string(str) {
let result = ""
for i in len(str)-1..0 {
result += str[i]
}
return result
}

Efficient string reversal using a loop.

List Operations

func filter_even(numbers) {
let result = []
for n in numbers {
if n % 2 == 0 {
result.push(n)
}
}
return result
}

Filter even numbers from a list.
